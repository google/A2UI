# A2UI Generator

This is a UI to generate and visualize A2UI responses.

## Prerequisites

1. [nodejs](https://nodejs.org/en)

## Running

1. Install the dependencies: `pnpm i`
2. Run the [A2A server](../../a2a_samples/a2ui_restaurant_finder/)
3. Run the dev server: `pnpm dev`
4. Open http://localhost:5173/
